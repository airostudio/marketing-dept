## Best Practices

### 1. Hook Viewers Early
- First 3 seconds determine success
- Start with the value proposition
- Use pattern interrupts

### 2. Optimize for Platform
- YouTube: 7-15 minutes, educational
- LinkedIn: 1-3 minutes, professional
- Instagram/TikTok: 15-60 seconds, entertaining
- Twitter: Under 45 seconds, punchy

### 3. Always Include Captions
- 85% of social videos watched without sound
- Improves accessibility
- Boosts SEO

### 4. Test Thumbnails
- A/B test different thumbnail styles
- Use high contrast and readable text
- Show faces when possible

### 5. Consistent Branding
- Use consistent colors and fonts
- Include logo placement
- Maintain brand voice
