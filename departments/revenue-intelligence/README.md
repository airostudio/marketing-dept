## Forecasting Methodology

### 4-Lens Forecast (Clari)
1. **Rep Forecast**: What reps submit
2. **Manager Forecast**: Manager's view
3. **AI Forecast**: Machine learning prediction
4. **Historical**: Based on past performance

Combined for most accurate forecast.

### Pipeline Categories
- **Commit**: 90%+ confident
- **Best Case**: 70-89% confident
- **Pipeline**: 50-69% confident
- **Omitted**: <50% or pushed

### Weekly Forecast Rhythm
- Monday: Reps submit
- Tuesday: Manager review
- Wednesday: AI analysis
- Thursday: Leadership review
- Friday: Final forecast to exec team
