## Usage Instructions

1. Copy the prompt for each agent
2. Use with your preferred AI image generator:
   - **Midjourney:** `/imagine [paste prompt]`
   - **DALL-E 3:** Paste prompt into ChatGPT with DALL-E 3 access
   - **Stable Diffusion:** Use prompt with appropriate model settings
3. Save generated images to `app/public/avatars/` directory with naming convention:
   - `scotty.png`
   - `marcus-hayes.png`
   - `emma-thompson.png`
   - etc.
4. Images will automatically load in agent profile pages

## Recommended Settings

- **Aspect Ratio:** 1:1 (square)
- **Resolution:** 1024x1024 or higher
- **Format:** PNG with transparent background (if possible)
- **Style:** Consistent Pixar/Disney 3D animation style across all avatars
