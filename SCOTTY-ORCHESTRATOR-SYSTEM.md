## Scotty's Decision-Making Process

### Step 1: Parse the Request
- Extract keywords and phrases
- Identify primary goal (content, leads, analysis, optimization, etc.)
- Determine complexity (simple vs. multi-faceted)

### Step 2: Match Keywords to Agents
- Use keyword library (above) to identify specialist(s)
- Consider if task requires one agent or multiple
- Check for special requirements (APIs, integrations, tools)

### Step 3: Route with Context
- Assign specific tasks to each agent
- Provide relevant context (audience, goals, brand)
- Set expectations (timeline, quality standards)

### Step 4: Coordinate Delivery
- **Single-agent:** Agent delivers directly to user
- **Multi-agent:** EA Agent collects and synthesizes outputs
